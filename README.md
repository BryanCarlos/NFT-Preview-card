### What I learned

For this project I tried to do a better CSS, and I learn how to use the "&" inside the CSS class.

Here's an example:

```html
    <h2>
      Equilibrium #3429
    </h2>
```
```css
     h2 {
      color: var(--white);
      transition: all .2s;

      &:hover {
        color: var(--cyan);
  }
}
```
